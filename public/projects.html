Hey Coach!

An all purpose, well designed fitness web application that features user authentication and personal data display to track nutrition and exercise information.
This was a group project from my Software Engineering course. My team and I used React for the front end, Firebase for user authentication, and other Javascript UI libraries such as MUI and Chartjs.
We finished a backend database and created an API to utilize the personalized data, but never got to fully implement it into the app.

Featured: The landing page for new users, which I designed as well as the log in/sign up pages using React-Bootstrap.

Cipher

An instant messaging application with a heavy focus on privacy and security. The main features include end-to-end encryption of all messages and non-persistence of messages so they are deleted once read.

Technologies: Ejabberd (xmpp server framework), Java (encryption algorithms and Android app), Swift (IOS app), React (web app).

Featured: The advertisement me and my group put together for our app; I am the speaker here.
