What is a split?

When it comes to lifting weights, there are an overwhleming number of exercises to choose from. Dividing them up into groups and creating whats called a "split", will help to maximize the effectiveness of any exercises you choose. A split will usually select specifics muscles or groups of muscles to target on certain days. For example, the most primitive split would be to designate each muscle to its own day.
Let's look at this sort of split:

Monday Tuesday Wednesday Thursday Friday Saturday Sunday
Chest Back Legs Shoulders Triceps Biceps REST

The issue with this split is that you are only hitting each muscle group once a week. So after you work your chest, it will recover and then sit around waiting to be worked again next week. This is not efficient to build muscle since you are essentially wasting time while certain muscles are already fully recovered, but not getting reworked.

Push Pull Legs

The most commmon split today at least amongst teenagers and young adults is "Push Pull Legs". The method with this type of split is to fit all exercises into 3 separate days in order to hit them twice a week. All pushing related exercises (usually these are chest, triceps and shoulders lifts) are done in one day, pull exercises (usually back and biceps) are another day, and then legs would be the last day.
Let's see how this split would look in a week:

Monday Tuesday Wednesday Thursday Friday Saturday Sunday
Push Pull Legs Push Pull Legs REST

You can see that this split requires the same six days a week commitment, but is much more efficient as muscles will recover and then quickly be reworked.
