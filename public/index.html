This project is a collection of all of my knowledge and experience in the field of software engineering all meshed into one. I wanted to have the quality of this website speak to my skills, as well as provide an easy flow to link projects that could further demonstrate my abilities. This website alone is just a node.js + express.js application with raw HTML, because I do not need any truly interactive features for this static page.

All of the styling and layout was achieved with Bootstrap except for the buttons on this page and contact page. The link buttons, is the name I gave these buttons, exist to enhance usabaility when I want to create a more aesthetic link.

I am hosting this website using Amazon Web Service's EC2 instance to run the express server on.

The Git repository I utilize for this project is cloned into the EC2 instance so that I can easily pull and deliver production updates.
